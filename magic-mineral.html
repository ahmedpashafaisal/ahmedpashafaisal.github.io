MAGIC MINERAL (ONGOING)

DESCRIPTION

Explore the dangerous cave with very limited dimming light and experience the magic that both helps and haunts you. A top-down 2D survival game where you need to find a magic crystal that can potentially save your village.


YEAR

2023


GENRE

2D Top-Down Survival


PLATFORM

Web / PC

CONTRIBUTION

Game Design

This is an ongoing project but the overall design of the enemies and player abilities have been finalised. The Narrative Design is still in progress as the game demands a Branching Narrative to paint the player psychology effectively.

  • Created and updated concept
  • Created Game Design Document
  • Enemy Behavior and Player Ability Design
  • Narrative Design

Game Art

This was the most interesting and my favourite part of the contribution. I wanted to maintain a unique look to the overall game and create the scary dungeon vibe with the lights and darkness.

  • Environment Design
  • Character design and Animation
  • Shader Creation using Unity Shader Graph
  • Tilemap Creation

Programming (Unity)

Programming wise this is a very demanding job considering the scale of the game. I have already designed and developed the main systems but need to implement the rest of the enemies, and an advanced dialogue system for the Branching Narratves

  • AI Player Chase System
  • Ability System
  • Modular Architecture using Scriptable Object
  • Event System
  • Basic Dialogue System
