THE UNWANTED

DESCRIPTION

Can you take the helpless ghost back to the forest and pass all the challenges on the way? A point-and-click adventure game made as a project for the Advance Game Development course in the Digital Learning Games program, at Tallinn University.

YEAR

2023


GENRE

Point & Click Adventure


PLATFORM

Web

CONTRIBUTION

Game Design

The main objective of this game was to design a point and click adventure with puzzle element. The narrative of this game is very close to me.

  • Concept and narrative design
  • Character and ghost follow mechanics
  • UI Design

Game Art

The game art was developed from scratch and I tried to maintain a stylized and personal look to the overall game as the game is trying to tell an emotional story. That's why I have chosen te hand drawn style

  • Hand drawn Environment Design
  • Character design and Animation
  • Unity Post Processing using URP

Programming (Unity)

Implementing the logic for this game was very simple, but the requirement was to make the game persistent through scenes. I have learned some advanced architecture during this game.

  • Scene Persistance
  • Character and Ghost Mechanics implementation
  • Dialogue System
  • Event System
